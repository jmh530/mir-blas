# mir-blas
NDSLICE wrapper for BLAS

## Required static libraries
 - `blas`/`cblas` - CBLAS API

## Wrapped API

### Level 1
 - dot
 - nrm2
 - asum
 - copy
 - axpy
 - scal
 - swap

### Level 2
 - gemv
 - ger

### Level 3
 - gemm
 - syrk
 - trmm
 - trsm
